# Analýza projektu – Simulátor križovatky

Tento priečinok obsahuje analytickú dokumentáciu projektu **Simulátor križovatky so semaformi**.

## Obsah

- `analyza.md` – hlavná business analýza
- `stavovy_automat.md` – analýza stavového automatu semaforov
- `api.md` – návrh REST API
- `websocket.md` - návrh websocket
- `simulacia_vozidiel` - logika premávky
- `datovy_model` - úložisko dát systému
- `ui.md` – analýza používateľského rozhrania
- `use_cases.md` – prípady použitia
- `diagramy.md` – doplnkové Mermaid diagramy a návrhy
